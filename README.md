✨ **`xuender/grephub` is a GUI that supports multiple popular search tools.**

- [ripgrep (rg)][rg-url]
- [ugrep (ug)][ug-url]
- [The Silver Searcher (ag)][ag-url]

## 🚀 Install

```shell
go install github.com/xuender/grephub@latest
```
